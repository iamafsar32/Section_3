# Employee CRUD Operation using Python and MongoDB

## Project Description
This project is a simple Employee Management System developed using Python and MongoDB.

It performs CRUD operations:

- Create Employee
- Read Employee Details
- Update Employee Details
- Delete Employee Details

## Technologies Used

- Python
- MongoDB
- PyMongo

## Requirements

Install package:

pip install pymongo

## Database Details

- Database Name: employee_db
- Collection Name: employees

## How to Run

1. Start MongoDB server

mongod

2. Run Python file

python employee.py

## Features

- Menu Driven Program
- Insert Employee Records
- View All Employees
- Update Existing Employee
- Delete Employee Record
