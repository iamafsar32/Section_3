# Product CRUD Operation using Python and MongoDB

## Project Description
This project is a simple Product Management System developed using Python and MongoDB.

It performs CRUD operations:

- Create Product
- Read Product Details
- Update Product Details
- Delete Product Details

## Technologies Used

- Python
- MongoDB
- PyMongo

## Requirements

Install package:

pip install pymongo

## Database Details

- Database Name: product_db
- Collection Name: products

## How to Run

1. Start MongoDB server

mongod

2. Run Python file

python product.py

## Features

- Menu Driven Program
- Insert Product Records
- View All Products
- Update Existing Product
- Delete Product Record
